# 产品需求文档 (PRD) - 山西方言Emoji猜词（晋在Emoji）

## 1. 项目概述
- **项目名称**：山西方言Emoji猜词（晋在Emoji）
- **一句话简介**：看山西方言词条，从四个表情组合里挑出对的——10 道题，8 秒一道，测测你是不是正宗山西人。
- **目标平台**：抖音互动空间
- **交付形态**：纯离线 HTML5 单文件（`index.html`），总体积小于 8MB，无外部资源依赖。

## 2. 核心玩法流程
1. **开场画面（3 秒）**
   - 屏幕中央大字：🏮 山西方言大挑战 🏮
   - 副标题：看方言，选 Emoji，你是正宗老醯儿吗？
   - 底部按钮：开始挑战
   - 背景与风格：暖黄色渐变（类似老式账本或宣纸的颜色），四角有简单的 CSS 窗花装饰。

2. **答题界面**
   - 题号展示：显示当前题号（如 第 3/10 题）。
   - 发音按钮：点击后使用 Web Speech API 朗读当前方言。
   - 词条展示：中央大字显示方言词条（48px）。
   - 倒计时条：8秒总长，动态减少，颜色从绿到黄再到红。
   - 选项卡片：4个包含 Emoji 的选项卡片，单选。

3. **答题反馈（1.5 秒停留）**
   - **答对**：选中项背景变绿，显示 ✅。弹出释义卡片，展示普通话释义和例句。
   - **答错**：选中项背景变红，显示 ❌。正确选项高亮为绿色。弹出释义卡片。
   - **超时**：所有选项变灰，提示“⏰ 时间到！”，高亮正确选项并弹出释义卡片。
   - *反馈期间屏蔽点击操作，1.5 秒后自动进入下一题。*

4. **结果结算**
   - 展示总分（X/10）及对应称号（10分：正宗老醯儿；7-9分：半个山西人；4-6分：还需修炼；0-3分：纯外地人）。
   - 列表回顾每题的词条、玩家选择、正确答案及对错标记。
   - 底部按钮：再来一次、生成成绩卡（分享）。

5. **分享卡片生成**
   - 使用离线 Canvas 绘制长图（750x1334），包含标题、得分、称号及答题详情。
   - 渲染为图片供玩家长按保存。

## 3. UI/UX 设计要求
- **美术风格**：暖黄色调，现代简洁，卡片式布局，类似“知识答题”小程序。
- **布局适配**：竖屏自适应，使用 viewport + rem + flexbox 确保无横向滚动条，适配主流手机。
- **动效要求**：平滑的颜色过渡，进度条平滑缩减，卡片弹出动画。

## 4. 题库要求
- 内置不少于 15 题的方言题库（包含：圪蹴、闹他、寡气、日能、扑死砍活、闹不机密、真日塌、得劲儿、逛荡、捞摸、不赖、胡咧咧、磕膝盖、夜儿个、后生等）。
- 干扰项设计原则：近义干扰、反义干扰、随机干扰，保证一定的迷惑性。
- 每次游戏随机抽取 10 题，题目内的 4 个选项顺序随机打乱。
